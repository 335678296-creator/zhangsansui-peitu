# 真实物件选择与原创隐喻

## 原则

真实物件是隐喻的载体。选物件的标准不是"好看"，而是**它能不能天然承载这个处境，并让张三岁对它做一个物理动作**。

## 好物件 vs 坏物件

**好物件**（真实、轻、能承载动作）：

- 线缆、绳子、长尾夹、回形针、胶带、便利贴、纸条、纸堆。
- 笔记本电脑、手机（压成中性深色，缩小）、放大镜、筛子、漏斗、夹子、闹钟、印章、工牌、挂绳。
- 沙漏、纸箱、滑轮、磁铁、订书机、量尺。

**坏物件**（解释性、堆砌、截图感）：

- App 界面、聊天框截图、仪表盘、图表、UI 卡片。
- 一堆 Logo、品牌物料、商业 KV。
- 复杂机械、写实办公室、城市场景。

## 物件的中性色约束

张三岁是画面唯一暖色焦点。所以：

- 真实物件用牛皮纸、白、灰、原木、金属、深色。
- **不要让物件是粉色/暖红大面积**，否则和张三岁糊在一起。
- 红色只给"报警/痛点"做小面积情绪钉子（如一根红线、一个红灯）。

## 从抽象到物理动作（隐喻转译表）

| 抽象处境 | 物理动作 | 真实物件 |
| --- | --- | --- |
| 被会议拉走 | 被线往里拽，后仰蹬腿 | 线缆 + 笔记本 |
| 消息太多回不过来 | 举板挡住涌出的纸条 | 手机 + 纸条流 + 纸板 |
| 线上出故障 | 蹲着解打结的线 | 缠成结的线 + 红灯 |
| 反复返工 | 放大镜下圈红 | 放大镜 + 纸堆 + 红笔 |
| 被 AI 重新定义 | 拽住被贴标签的工牌 | 工牌挂绳 + "AI"贴纸 |
| 简历被刷 | 把纸推进筛子 | 筛子/漏斗 + 纸堆 |
| 时间被压 | 被沙漏压住/扛沙漏 | 沙漏 |
| 任务排队 | 被夹子一个个夹住 | 长尾夹队列 |

## 反复刻规则

- 隐喻每次都要让**当前内容**重新决定，不要不看内容直接套旧物件组合。
- 母版的物件可以在内容匹配时复用，但**空间关系或张三岁动作至少要换一个**。
- 机械复刻 = 直接套用旧图的物件、摆放、姿态、标签结构，只换文字。这是最常见的失败，必须避免。

## 原创隐喻自检

- 这个物件，普通人一眼知道它是干嘛的吗？（要"真实可识别"，不要生造装置）
- 张三岁对它的动作，是不是一个清楚的物理动作？
- 删掉张三岁，画面还成立吗？成立 = 失败（张三岁太装饰）。
- 一眼看上去，是"真实物件小现场"还是"信息图/教程页"？后者 = 重做。
