# 张三岁 IP（开源内核版）

## 角色定义

张三岁是这套配图的固定 IP。它不是死板 Logo，也不是表情包，是一个在真实物件现场里认真做事的可爱小团子。

它可以因场景不同有细微形体差异：高一点、矮一点、被拉时瘦一点、被压时扁一点。变化要小，**识别必须稳定**。

## 双层身份系统

本开源版只使用 **🔒 文字内核**——5 个任何文生图模型都能收敛的核心特征。这样任何人用任何模型、纯文字也能生出认得出的张三岁，不依赖参考图、不依赖任何人的 API 额度。

> ✨ 满配细节（帽顶小兔脸、黄色爱心发夹、肚子白色小熊 logo、精确发型）属于作者私有版的"高保真层"，不在开源内核内。开源版**不要**强行加这些细节——文生图复现不稳，加了反而画歪。

## 🔒 文字内核：5 个必留特征

每一条都要写进生图提示词，缺一不可：

1. **团子比例**——大头、小圆身、短粗四肢，矮胖 chibi。
2. **粉色兔耳连帽**——两只立起的圆兔耳的粉色连帽，是辨识度最高的剪影锚点。
3. **棕色齐刘海**——从帽檐下露出来，框住脸。
4. **极简脸公式**——两颗黑豆眼 + 粉色圆腮红 + 一条小弧嘴。不要大眼睛、不要复杂五官。
5. **渲染风格**——粗深色（深棕/深褐）描边 + 平涂纯色 + 无渐变 + 纯白干净背景。

只要这 5 条到位，出来就是张三岁。

## 张三岁必须承担核心动作

张三岁不能只站在旁边看。它要**用身体和真实物件发生物理接触**。

合格动作（保持完整形态去做）：

- 被一根线缆往笔记本/会议里拽，后仰、蹬腿。
- 双手举纸板挡住涌出来的消息纸条。
- 蹲下解开打结的报警线。
- 趴在放大镜下改稿。
- 拽住自己被贴上标签的工牌。
- 把简历纸往筛子里推。
- 被下班线缆往回拖。

不合格动作：

- 站在角落看物件。
- 举牌子解释主题。
- 对着物件摆拍、没有物理接触。
- 删掉张三岁后画面意思仍然完整。

判断标准：

```text
如果删掉张三岁，隐喻还完全成立，说明张三岁失败了。
```

## 形变规则（和小黑不同！）

张三岁细节比小黑多，**必须保持完整形态**，不能融成一团或丢掉识别特征。

允许：

- 轻微的后仰、前倾、被拉伸/压扁一点点。
- 半个身子躲在物件后面、被物件部分遮挡。
- 手臂用力、蹬腿、抱东西等姿势变化。

禁止：

- 融成一个抽象团子、丢掉兔耳帽。
- 把脸画复杂、加夸张表情或表情包五官。
- 穿西装、戴道具帽来解释身份。
- 每张都同一个姿势同一个比例（会死板）。

## 配色规则（和小黑不同！）

小黑是纯黑，靠纯白留白显高级。张三岁**自带粉色，它本身就是画面的"主色事件"**。

- **真实物件压成中性色**：牛皮纸、白、灰、原木、金属、深色。不要让物件也是粉色，否则粉上加粉糊成一片。
- **额外彩色点缀比小黑更克制**：每张 2-4 个小点缀，优先偏冷（蓝、黄），别再大面积上暖色。
- 张三岁的粉是画面唯一的"暖色焦点"，其他都给它让路。

## 性格

- 认真，不刻意卖萌（萌来自"认真做荒诞事"，不是表情）。
- 有点笨拙但不蠢。
- 像一个安静的小操作员，在荒诞现场里很冷静。
