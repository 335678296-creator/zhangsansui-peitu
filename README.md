# 张三岁 实物场景图 · Zhangsansui Scenes

> 把中文内容里的处境、压力、转折和项目故事，变成"张三岁 + 真实物件 + 物理动作 + 留白叙事"的正文配图。
>
> 16:9 正文图 | 彩蛋长卷故事图 | 真实物件小现场 | 张三岁 IP | Claude / Codex Skill

---

## 这个仓库是什么

一个内容配图 Skill，指导 AI Agent 为中文文章、帖子、教程、案例、项目复盘和个人经历，生成"张三岁"视觉配图。

核心公式：

```text
张三岁 + 真实物件 + 物理动作 + 短中文标签 + 留白叙事
```

一句话：**让读者先看到一个真实、轻、可爱的小现场，再在 1 秒内意识到"这说的就是我"。**

> 致敬 [Ian Xiaohei Scenes](https://github.com/helloianneo/ian-xiaohei-scenes) 的方法论。本项目把固定 IP 从"小黑"换成了"张三岁"，并针对可爱角色重做了形变与配色规则。

---

## 张三岁是谁

张三岁是一个在真实物件现场里认真做事的可爱小团子。本开源版用 **🔒 文字内核** 定义它——5 个任何文生图模型都能复现的核心特征：

| # | 特征 |
| --- | --- |
| 1 | 团子比例（大头小圆身、短粗四肢） |
| 2 | 粉色兔耳连帽（两只立起圆兔耳） |
| 3 | 棕色齐刘海（帽檐下露出框脸） |
| 4 | 极简脸（黑豆眼 + 粉腮红 + 小弧嘴） |
| 5 | 粗深褐描边 + 平涂 + 纯白底 |

把这 5 条写进任意文生图模型的提示词，就能得到认得出的张三岁。

---

## 重要：这是"文字可复现"的开源版

和小黑那种纯黑剪影不同，可爱角色细节多、复现难。所以本项目专门做了 **文字内核**，让它能像小黑一样"纯文字复现"：

- ✅ **纯文字即可用**：任何文生图模型（gpt-image / Nano Banana / 即梦 Seedream / SD / Midjourney…），把内核 5 特征写进提示词就行。
- ✅ **零 API key、零额度依赖**：你用你自己的图像工具，不碰任何人的账号。
- ✅ **可选参考图**：仓库自带 `assets/zhangsansui-kernel-ref.png` 内核定妆图，模型支持图生图时可喂进去提高一致性，但**不是必须**。

> 注意：内核版**不含**作者私有版的高保真细节（帽顶兔脸、爱心发夹、肚子小熊 logo）。那是作者的精装层，不在开源范围内。开源版请不要强加这些细节——文生图复现不稳。

## 两种模式

### 标准模式（16:9 正文配图）

纯白干净背景 / 一个真实物件小现场 / 一个核心物理动作 / 张三岁承担动作 / 2-4 个中文短标签 / 真实物件压中性色 / 少量冷色点缀。

### 彩蛋长卷模式

超横版长卷故事图 / 高级近白背景 / 一条手绘曲线 / 5-8 个真实物件节点 / 每个节点张三岁参与 / 左起右收 / 节点不编号、不做 PPT 流程图。用于个人经历、项目复盘、产品演化、成长路径。

## 安装

### Claude Code

把 skill 子目录拷进 Claude Code 的 skills 目录：

```bash
git clone https://github.com/<your-name>/zhangsansui-peitu.git
cp -R zhangsansui-peitu/zhangsansui-peitu ~/.claude/skills/
```

### Codex

```bash
mkdir -p "${CODEX_HOME:-$HOME/.codex}/skills"
cp -R zhangsansui-peitu/zhangsansui-peitu "${CODEX_HOME:-$HOME/.codex}/skills/"
```

安装后调用：

```text
用 zhangsansui-peitu，为这篇中文文章设计并生成几张张三岁实物场景正文配图。
```

## 怎么用

### 只做配图规划

```text
用 zhangsansui-peitu，先不要生图。
分析下面这篇文章哪里适合做"张三岁 + 真实物件 + 物理动作"的正文配图，输出 5 张左右的 shot list。
<粘贴文章>
```

### 生成正文配图

```text
用 zhangsansui-peitu，把下面这篇文章生成 4 张张三岁实物场景配图。
要求：16:9、纯白背景、真实物件压中性色、张三岁承担核心动作、≤3 个中文短标签。
<粘贴文章>
```

> 实际出图用**你自己的图像模型/工具**。Skill 负责理解内容、锁母版、写提示词、做 QA；图由你的模型生成。

## 注意事项

- 每张标准图只讲一个核心物理动作。
- 中文标签越短越稳（文生图中文易出错字）；糊了减词重生成。
- 张三岁必须承担核心动作；删掉它画面仍完整 = 它太装饰。
- 示例图是质量母版，不是可描摹模板。
- 真实物件压中性色，别用粉色（会和张三岁糊）。
